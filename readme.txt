Random WordPree Theme Plugin load random themes located in wp-content/themes automatically everytime visitor open the website.

== Description ==

Auto Random WordPress Theme located in wp-content/themes . This random theme plugin will load different theme every time your visitor come to your website / refresh your website. Just make sure , you have installed some perfect themes. At least you must have 2 WordPress Theme in wp-content/themes to show different everytime website load.
